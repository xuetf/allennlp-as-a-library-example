## Project Structure

- data：各任务训练、验证、测试数据等。

- experiments：各任务训练的.json配置文件，训练的核心入口文件。

- librarys: 各任务代码，主要包括数据读取/模型/预测器(对外提供服务时使用)

- tests: 各任务的单元测试代码。
    
- outputs：各任务训练时的输出，例如checkpoint，vocabulary等。在线服务,tensorboard查看等都需要基于该文件夹。
   
- scripts: 各任务实际的训练/测试/服务shell脚本，直接运行即可。
